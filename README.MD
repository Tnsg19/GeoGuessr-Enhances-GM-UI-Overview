# GeoGuessr Duels Browser Extension

## Setup Instructions

### Installation
1. Clone or download this repository
2. Open your browser and navigate to the extensions page:
	- **Chrome/Edge**: `chrome://extensions`
	- **Firefox**: `about:addons`
3. Enable "Developer Mode" (top right)
4. Click "Load unpacked" and select this folder

### Configuration
- Adjust settings in `config.js`

### Usage
- Click the extension icon to activate
- Start a GeoGuessr Duel match
- The extension will provide enhanced features and statistics

### Requirements
- Modern browser (Chrome, Edge, or Firefox)
- Active GeoGuessr account

### Recommend
- Use a desktop or laptop with 1920x1080 resolution
